# Plantillas para Nuevas Habilidades

Usa estas plantillas como punto de partida para crear archivos `SKILL.md`.

## Plantilla Base para SKILL.md

```markdown
---
name: nombre-de-tu-habilidad
description: Describe aquí de forma concisa qué hace esta habilidad y por qué es útil.
---

# Nombre de la Habilidad

## Cuándo usar esta habilidad

- [Situación 1]
- [Situación 2]

## Cómo usar esta habilidad

1. **Paso 1**: [Descripción]
2. **Paso 2**: [Descripción]

## Instrucciones Especiales

- [Regla o consejo importante]
```

## Ejemplo de YAML para la descripción

La descripción es vital para que el agente sepa cuándo activar la habilidad. Ejemplo:
`description: Asiste en la optimización de imágenes WebP para mejorar el rendimiento del sitio Astro.`
